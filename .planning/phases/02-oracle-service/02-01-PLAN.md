<objective>
Add a `set_arweave_txid` instruction to the Anchor program so the oracle can write Arweave transaction IDs back to resolved PixelState accounts after uploading prompts to Arweave.

Purpose: The PixelState struct has `arweave_txid: [u8; 43]` and `has_arweave_txid: bool` fields, but no instruction exists to set them. The oracle needs this instruction to complete the commit-reveal chain (prompt hash at open -> prompt text on Arweave -> txid on-chain). Without it, ORC-05's Arweave write-back cannot work.

Output: Updated Anchor program with `set_arweave_txid` instruction, passing tests, rebuilt IDL, redeployed to devnet.
</objective>
