2. **WalletDisconnectBanner** (`app/src/components/ui/WalletDisconnectBanner.tsx`):
   - "use client" directive.
   - Props: none (self-contained, reads wallet state).
   - Watches `useWallet().connected` transitions via `useRef`:
     ```
     const { connected, connect } = useWallet();
     const prevConnectedRef = useRef(connected);
     const [showBanner, setShowBanner] = useState(false);

     useEffect(() => {
       if (prevConnectedRef.current === true && connected === false) {
         setShowBanner(true);
       }
       if (connected) {
         setShowBanner(false);
       }
       prevConnectedRef.current = connected;
     }, [connected]);
     ```
   - When `showBanner` is true, render a fixed top banner (below header, ~56px from top):
     a. Background: warning orange (#ED8936) with 90% opacity
     b. Text: "Wallet disconnected. Reconnect to continue playing."
     c. "Reconnect" button that calls `connect()` — styled in white with rounded corners
     d. "Dismiss" X button to hide the banner
   - Banner uses motion slide-down animation.

   - Props: none. This is a static UI element per FS-11.
