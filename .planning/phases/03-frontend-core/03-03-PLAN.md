       - useEffect with 1-second interval:
         - elapsed = Math.floor(Date.now() / 1000) - openedAtSeconds
         - remaining = Math.max(0, ROUND_DURATION_SECONDS - elapsed)
         - setSecondsLeft(remaining)
       - Derived: isLocked = secondsLeft <= LOCKOUT_SECONDS (120)
       - Derived: isFinalDrama = secondsLeft <= 10 && secondsLeft > 0
       - Derived: minutesDisplay = String(Math.floor(secondsLeft / 60)).padStart(2, "0")
       - Derived: secondsDisplay = String(secondsLeft % 60).padStart(2, "0")

       - Reads pixelPDA and seasonNumber/pixelIndex from store (to derive PDA)
       - Uses useAnchorProgram() for account fetch
       - fetchPools: program.account.pixelState.fetch(pixelPDA) -> extract colorPools (BN[]) and totalPool
       - Convert each BN to number via .toNumber() (safe for SOL amounts under 9007 SOL)
       - Update store via setPixelState with new colorPools/totalPool
       - On mount: fetchPools()
       - Set interval: 60_000ms (POOL_REFRESH_INTERVAL_MS) — THIS IS INTENTIONAL ANTI-SNIPING, DO NOT REDUCE
       - On document.visibilitychange to "visible": immediately fetchPools() and reset the 60s interval
       - Cleanup: clear interval on unmount
       - Return { computeMultiplier, computePoolPercent } utility functions:
         - computeMultiplier(colorPool, totalPool): if either is 0, return "---"; else return `${((totalPool * 0.95) / colorPool).toFixed(1)}x`
         - computePoolPercent(colorPool, totalPool): if totalPool is 0, return "0%"; else return `${((colorPool / totalPool) * 100).toFixed(0)}%`

    3. Create `app/src/hooks/usePlaceBet.ts` ("use client"):
       - Uses useAnchorProgram(), useWallet() (for publicKey), useGameStore (for seasonNumber, currentPixelIndex)
       - State: { isSubmitting: boolean, error: string | null }
       - placeBet(colorIndex: number, amountSol: number):
         a. Guard: if isSubmitting, return (prevents double-submit — Pitfall 6)
         b. Guard: if !program || !wallet.publicKey, return
         c. Set isSubmitting = true
         d. Derive PDAs: pixelPDA, betPDA, statsPDA using lib/pda functions
         e. Convert amount: new BN(Math.floor(amountSol * LAMPORTS_PER_SOL))
         f. Call program.methods.placeBet(colorIndex, amount).accounts({...}).rpc({ commitment: "confirmed" })
         g. On success: setPlayerBet({ colorIndex, amount: amountSol * LAMPORTS_PER_SOL, claimed: false })
         h. Set isSubmitting = false
         i. On error: set error message, set isSubmitting = false
       - Return { placeBet, isSubmitting, error, clearError }
