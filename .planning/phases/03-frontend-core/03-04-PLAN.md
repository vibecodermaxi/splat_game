    1. Create `app/src/hooks/useResolution.ts` ("use client"):
       - Watches store: activePixelData (specifically status field) and playerBet
       - Uses useRef to track previous status — detects transition from "locked" (or "open") to "resolved"
       - When transition detected:
         a. Set `resolvedPixel` state: { pixelIndex, winningColor, winningColorHex }
         b. Determine win/loss: if playerBet && playerBet.colorIndex === resolvedPixel.winningColor -> win
         c. If win: compute estimated payout = (playerBet.amount / colorPools[winningColor]) * totalPool * 0.95
         d. Set `resolution` state: { type: "win" | "loss", winningColor, winningColorName, payoutSol?: number }
         e. After animation duration (2.5s total): clearForNewRound() in store
       - Return { resolution, resolvedPixel, clearResolution }

    2. Create `app/src/components/canvas/ResolutionAnimation.tsx` ("use client"):
       - Props: { pixelIndex: number, winningColorHex: string, onComplete: () => void }
       - Rendered as an overlay positioned exactly over the resolved pixel cell in the canvas
       - Use Motion:
         - Radial burst: `clipPath: "circle(0% at 50% 50%)"` to `"circle(150% at 50% 50%)"`, duration 0.2s, easeOut
         - Glow: `boxShadow` from `0 0 0 0 ${color}80` to `0 0 12px 4px ${color}80` to `0 0 0 0`, duration 0.5s
       - AnimatePresence for mount/unmount transitions
       - After animation completes: call onComplete
       - prefers-reduced-motion: skip clip-path animation, instant color fill

    3. Create `app/src/components/ui/WinNotification.tsx` ("use client"):

         - "SPLAT! You nailed it." in Fredoka One, white

         - Display as "+{amount} SOL" in Nunito Bold, Splat Green
