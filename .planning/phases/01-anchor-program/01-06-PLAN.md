   ```bash
   solana-keygen new --outfile keys/treasury-wallet.json --no-bip39-passphrase --force
   solana-keygen new --outfile keys/jackpot-wallet.json --no-bip39-passphrase --force
   ```
   Create the `keys/` directory first. Add `keys/` to `.gitignore` if not already present (keypair files must NOT be committed).

2. Extract the public keys from the generated keypairs:
   ```bash
   solana address -k keys/treasury-wallet.json
   solana address -k keys/jackpot-wallet.json
   ```

3. Update `programs/pixel-predict/src/constants.rs`:
   - Replace the TREASURY_WALLET placeholder (`11111111111111111111111111111112`) with the real treasury pubkey
   - Replace the JACKPOT_WALLET placeholder (`11111111111111111111111111111113`) with the real jackpot pubkey
   - Update the comments: change "REPLACE BEFORE MAINNET DEPLOYMENT" to "Devnet treasury wallet" and "Devnet jackpot wallet" respectively
   - Keep the "NOT admin-redirectable" comment on both

4. Update `Anchor.toml`:
   - Add a `[programs.devnet]` section with the same program ID: `pixel_predict = "FaVo1qzkbVt1uPwyU4jRZ7hgkJbYTzat8iqtPE3orxQG"`
   - Do NOT change the `[provider]` section (leave cluster as localnet for local development -- deployment will use --provider.cluster devnet flag)

5. Rebuild the program to verify the new constants compile:
   ```bash
   anchor build
   ```

   ```bash
   anchor test --skip-deploy
   ```
