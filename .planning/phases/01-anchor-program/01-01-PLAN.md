  <action>
Create all account state structs and enums. These are the contracts that all instruction plans will implement against.

**Enums** -- define in state/mod.rs with `#[derive(AnchorSerialize, AnchorDeserialize, Clone, PartialEq, Eq, InitSpace)]`:

- `SeasonStatus { Active, Completed, Intermission }` -- use `#[repr(u8)]` so it's 1 byte
- `RoundStatus { Open, Locked, Resolved }` -- use `#[repr(u8)]` so it's 1 byte

**State structs** -- all must derive `#[account]` and `#[derive(InitSpace)]`:

1. `state/config_account.rs` -- ConfigAccount (singleton PDA, seeds: `[b"config"]`):
   - `admin: Pubkey` (32 bytes) -- set at init, admin authority
   - `oracle: Pubkey` (32 bytes) -- rotatable by admin (named `oracle` not `oracle_pubkey` for Anchor `has_one` compatibility)
   - `bump: u8` (1 byte)

2. `state/season_state.rs` -- SeasonState (PDA seeds: `[b"season", season_number.to_le_bytes()]`):
   - `season_number: u16`
   - `grid_width: u8` (always 10 for v1)
   - `grid_height: u8` (always 10 for v1)
   - `current_pixel_index: u16` (0-99)
   - `status: SeasonStatus` (enum: Active, Completed, Intermission)
   - `total_volume: u64` (lamports)
   - `total_bets: u32`
   - `unique_wallets: u32`
   - `created_at: i64`
   - `completed_at: Option<i64>`
   - `bump: u8`

3. `state/pixel_state.rs` -- PixelState (PDA seeds: `[b"pixel", season_number.to_le_bytes(), pixel_index.to_le_bytes()]`):
   - `season_number: u16`
   - `pixel_index: u16` (0-99)
   - `x: u8`
   - `y: u8`
   - `status: RoundStatus` (enum: Open, Locked, Resolved)
   - `color_pools: [u64; 16]` (128 bytes)
   - `total_pool: u64`
   - `winning_color: Option<u8>` (0-15)
   - `shade: Option<u8>` (0-100)
   - `warmth: Option<u8>` (0-100)
   - `prompt_hash: [u8; 32]` (SHA-256 of pre-committed prompt)
   - `arweave_txid: [u8; 43]` (base58 Arweave txId, zeroed = not set)
   - `has_arweave_txid: bool`
   - `vrf_resolved: bool` (true if VRF fallback used)
   - `opened_at: i64`
   - `locked_at: Option<i64>`
   - `resolved_at: Option<i64>`
   - `bump: u8`

   For `arweave_txid`, since InitSpace doesn't handle `Option<[u8; 43]>` natively, store as `arweave_txid: [u8; 43]` (zeroed = not set) + `has_arweave_txid: bool`.

4. `state/bet_account.rs` -- BetAccount (PDA seeds: `[b"bet", season_number.to_le_bytes(), pixel_index.to_le_bytes(), player.key()]`):
   - `player: Pubkey` (32 bytes)
   - `season_number: u16`
   - `pixel_index: u16`
   - `color: u8` (0-15)
   - `amount: u64` (lamports)
   - `claimed: bool`
   - `bump: u8`

5. `state/player_season_stats.rs` -- PlayerSeasonStats (PDA seeds: `[b"stats", season_number.to_le_bytes(), player.key()]`):
   - `player: Pubkey` (32 bytes)
   - `season_number: u16`
   - `total_bets: u32`
   - `total_volume: u64` (lamports)
   - `correct_predictions: u16`
   - `colors_bet: [u16; 16]` (count per color, 32 bytes)
   - `bump: u8`

**state/mod.rs** -- re-export all structs and enums.

Do NOT create lib.rs, constants.rs, errors.rs, or instructions/ yet -- that is Task 3.
